📋簡介

PromptForge(「我們」、「我們的」)尊重您的隱私權。本隱私權政策說明我們如何收集、使用、揭露和保護您在使用 PromptForge 行動應用程式(「應用程式」、「服務」)時的資訊。

📊資訊收集

我們收集的資訊

PromptForge 採用「隱私優先」的設計理念,我們僅收集必要的資訊以提供服務:

1. 本地儲存資訊

  • 提示詞專案資料: 您建立的專案、提示詞版本、標籤等資訊
  • 優化歷史記錄: 您的提示詞優化歷史
  • 應用程式設定: 您的偏好設定和應用程式配置

⚠️重要: 這些資訊完全儲存在您的裝置本地,我們不會上傳或儲存到任何遠端伺服器。

2. API 金鑰

  • OpenRouter API Key: 用於呼叫 AI 優化服務
  • 儲存方式: 使用系統安全儲存機制(iOS Keychain / Android Keystore)加密儲存
  • 使用範圍: 僅用於向 OpenRouter 發送優化請求

3. 我們不收集的資訊

  • ❌ 個人識別資訊(姓名、電子郵件、電話號碼等)
  • ❌ 位置資訊
  • ❌ 裝置識別碼
  • ❌ 使用分析資料
  • ❌ 廣告追蹤資訊

🎯資訊使用

我們如何使用您的資訊

  1. 提供核心服務
    • 儲存和管理您的提示詞專案
    • 執行提示詞優化功能
    • 提供版本控制和比對功能
  2. 改善服務品質
    • 當您主動提交反饋時,我們可能會收集您提供的資訊以改善應用程式

我們不會做的事

  • ❌ 出售您的資料給第三方
  • ❌ 用於廣告目的
  • ❌ 與其他公司分享(除非法律要求)
  • ❌ 追蹤您的使用行為

🔗第三方服務

OpenRouter API

當您使用提示詞優化功能時,您的提示詞內容會透過 OpenRouter API 發送至 AI 模型進行處理。

  • 服務提供商: OpenRouter (https://openrouter.ai)
  • 傳輸資料: 您的提示詞內容
  • 隱私政策: https://openrouter.ai/privacy
  • 資料保護: 傳輸過程使用 HTTPS 加密

⚠️重要提醒:

  • 您的 API Key 和提示詞內容僅在優化時傳送給 OpenRouter
  • 我們不會儲存或記錄傳送給 OpenRouter 的內容
  • OpenRouter 的資料處理政策請參閱其隱私權政策

反饋功能

當您使用應用程式內的反饋功能時:

  • 您可以選擇性地提供電子郵件地址以便我們回覆
  • 您可以選擇性地附加截圖或檔案
  • 這些資訊僅用於處理您的反饋,不會用於其他目的

🛡️資料安全

安全措施

我們採取以下措施保護您的資訊:

  1. 本地加密儲存
    • API Key 使用系統級安全儲存(Keychain/Keystore)
    • 專案資料使用 Hive 本地資料庫儲存
  2. 傳輸加密
    • 所有網路請求使用 HTTPS/TLS 加密
  3. 最小權限原則
    • 應用程式僅請求必要的系統權限
    • 不要求存取聯絡人、相機、位置等敏感權限(除非您主動使用反饋附件功能)

💾資料保留

  • 本地資料: 資料保留在您的裝置上,直到您刪除應用程式或手動清除資料
  • 反饋資料: 反饋資訊保留至問題解決後的合理期間
  • 無伺服器儲存: 我們不在伺服器上儲存您的任何個人資料或提示詞內容

✅您的權利

資料控制權

您對自己的資料擁有完全控制權:

  1. 存取權: 所有資料儲存在您的裝置本地,您可隨時存取
  2. 修改權: 您可以隨時修改或刪除您的專案和提示詞
  3. 刪除權: 解除安裝應用程式將刪除所有本地資料
  4. 匯出權: 您可以使用分享功能匯出您的提示詞內容

👶兒童隱私

PromptForge 不針對 13 歲以下兒童。我們不會故意收集 13 歲以下兒童的個人資訊。如果您是家長或監護人,並且發現您的孩子向我們提供了個人資訊,請聯繫我們,我們將採取措施刪除該資訊。

🔄隱私權政策變更

我們可能會不時更新本隱私權政策。我們將透過以下方式通知您任何變更:

  • 在應用程式內發布新的隱私權政策
  • 更新本頁面頂部的「最後更新日期」

重大變更將在生效前至少 30 天通知您。

🌍國際資料傳輸

PromptForge 採用本地優先的設計:

  • 您的資料主要儲存在您的裝置上
  • 僅在使用優化功能時,提示詞內容會傳送至 OpenRouter API
  • OpenRouter 可能在不同國家/地區處理資料,請參閱其隱私權政策

🍪Cookie 和追蹤技術

PromptForge 行動應用程式不使用 Cookie 或類似的追蹤技術。

📧聯絡我們

如果您對本隱私權政策有任何疑問或疑慮,請透過以下方式聯繫我們:

  • 應用程式內反饋: 使用應用程式內的反饋功能
  • 電子郵件: [您的聯絡電子郵件]

⚖️法律依據(適用於 GDPR)

如果您位於歐洲經濟區(EEA),我們處理您個人資料的法律依據為:

  1. 合約履行: 提供您要求的服務
  2. 合法利益: 改善和維護我們的服務
  3. 同意: 當您主動提供反饋時

🌟加州居民權利(CCPA)

如果您是加州居民,您享有以下權利:

  1. 知情權: 了解我們收集的個人資訊類別
  2. 刪除權: 要求刪除您的個人資訊
  3. 選擇退出權: 選擇退出個人資訊的銷售(我們不銷售個人資訊)

由於 PromptForge 不收集或儲存個人識別資訊,這些權利主要透過您對本地資料的完全控制來實現。

🚨資料洩露通知

雖然我們不儲存您的個人資料在伺服器上,但如果發生任何可能影響您資料安全的事件,我們將:

  1. 立即調查事件
  2. 在合理時間內通知受影響的使用者
  3. 採取措施防止未來發生類似事件

📝總結

PromptForge 的核心理念是「隱私優先」:

  • ✅ 本地儲存,不上傳個人資料
  • ✅ 最小化資料收集
  • ✅ 透明的資料使用政策
  • ✅ 您完全控制自己的資料
  • ✅ 不追蹤、不分析、不販售

我們相信,保護您的隱私是建立信任的基礎。
