📋 簡介

PromptForge（「我們」、「我們的」）尊重您的隱私權。本隱私權政策說明我們如何收集、使用、揭露和保護您在使用 PromptForge 行動應用程式（「應用程式」、「服務」）時的資訊。

📊 資訊收集

我們收集的資訊

PromptForge 採用「隱私優先」的設計理念，我們僅收集必要的資訊以提供服務：

1. 本地儲存資訊

  • 提示詞專案資料：您建立的專案、提示詞版本、標籤等資訊
  • 優化歷史記錄：您的提示詞優化歷史
  • 應用程式設定：您的偏好設定和應用程式配置

⚠️ 重要：這些資訊完全儲存在您的裝置本地，我們不會上傳或儲存到任何遠端伺服器。

2. API 金鑰

  • OpenRouter API Key：用於呼叫 AI 優化服務
  • 儲存方式：使用系統安全儲存機制（iOS Keychain / Android Keystore）加密儲存
  • 使用範圍：僅用於向 OpenRouter 發送優化請求

3. 我們不收集的資訊

  • ❌ 個人識別資訊（姓名、電子郵件、電話號碼等）
  • ❌ 位置資訊
  • ❌ 裝置識別碼
  • ❌ 使用分析資料
  • ❌ 廣告追蹤資訊

🎯 資訊使用

我們如何使用您的資訊

  1. 提供核心服務
    • 儲存和管理您的提示詞專案
    • 執行提示詞優化功能
    • 提供版本控制和比對功能
  2. 改善服務品質
    • 當您主動提交反饋時，我們可能會收集您提供的資訊以改善應用程式

我們不會做的事

  • ❌ 出售您的資料給第三方
  • ❌ 用於廣告目的
  • ❌ 與其他公司分享（除非法律要求）
  • ❌ 追蹤您的使用行為

🛡️ 資料安全

安全措施

我們採取以下措施保護您的資訊：

  1. 本地加密儲存
    • API Key 使用系統級安全儲存（Keychain/Keystore）
    • 專案資料使用 Hive 本地資料庫儲存
  2. 傳輸加密
    • 所有網路請求使用 HTTPS/TLS 加密
  3. 最小權限原則
    • 應用程式僅請求必要的系統權限
    • 不要求存取聯絡人、相機、位置等敏感權限

✅ 您的權利

資料控制權

您對自己的資料擁有完全控制權：

  1. 存取權：所有資料儲存在您的裝置本地，您可隨時存取
  2. 修改權：您可以隨時修改或刪除您的專案和提示詞
  3. 刪除權：解除安裝應用程式將刪除所有本地資料
  4. 匯出權：您可以使用分享功能匯出您的提示詞內容

📧 聯絡我們

如果您對本隱私權政策有任何疑問或疑慮，請透過以下方式聯繫我們：

  • 應用程式內反饋：使用應用程式內的反饋功能
  • 電子郵件：[您的聯絡電子郵件]

📝 總結

PromptForge 的核心理念是「隱私優先」：

  • ✅ 本地儲存，不上傳個人資料
  • ✅ 最小化資料收集
  • ✅ 透明的資料使用政策
  • ✅ 您完全控制自己的資料
  • ✅ 不追蹤、不分析、不販售

我們相信，保護您的隱私是建立信任的基礎。
